StealthLens

Descripción:
StealthLens es una herramienta forense que reconstruye la actividad de un atacante incluso si eliminó logs, comparando directorios y snapshots.

Uso básico:
Configura las rutas en el script y ejecútalo:

    python StealthLens.py

Cómo descargarla:
Clona el repositorio:

    git clone https://github.com/tuusuario/OffSec-Tools.git

Luego navega a StealthLens:

    cd OffSec-Tools/StealthLens

Advertencia:
Solo usar en auditorías autorizadas.
