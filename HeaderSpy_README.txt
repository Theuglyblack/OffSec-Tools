HeaderSpy

Descripción:
HeaderSpy es una herramienta de Web Security que escanea masivamente cabeceras HTTP para detectar configuraciones erróneas, cabeceras faltantes y vectores de ataque como XSS, CSRF, CORS y Clickjacking.

Uso básico:
Ejecuta con una lista de URLs para analizar cabeceras:

    python header_spy.py --input urls.txt --output report.html

Cómo descargarla:
Clona el repositorio:

    git clone https://github.com/tuusuario/OffSec-Tools.git

Luego accede a la carpeta HeaderSpy:

    cd OffSec-Tools/HeaderSpy

Advertencia:
Solo para análisis con autorización.
