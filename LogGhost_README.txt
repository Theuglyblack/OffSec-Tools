LogGhost

Descripción:
LogGhost es una herramienta para Blue Team / Forense que detecta procesos ocultos, cambios no registrados y actividades sin logs clásicos, ideal para análisis post-explotación.

Uso básico:
Ejecuta el script para detectar anomalías:

    python LogGhost.py

Cómo descargarla:
Clona el repositorio:

    git clone https://github.com/tuusuario/OffSec-Tools.git

Luego entra en la carpeta LogGhost:

    cd OffSec-Tools/LogGhost

Advertencia:
Uso solo en entornos legales y educativos.
